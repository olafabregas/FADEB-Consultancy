# Fadeb Consultancy

Professional recruitment and consulting platform built with React, Vite, and Tailwind CSS.

## 📌 Quick Start

```bash
npm install      # Install dependencies
npm run dev      # Start development server (http://localhost:3000)
npm run build    # Build for production
npm run preview  # Preview production build
```

## 📄 Pages

- **Home** - Hero carousel with services overview
- **Job Seekers** - CV upload, job search guidance, training
- **Employers** - Recruitment services, partnerships, staff training
- **Services** - Recruitment, Training, CV/Resume services
- **About** - Company information and FAQ
- **Contact** - Contact form and information
- **Terms & Privacy** - Legal pages

## 🛠 Tech Stack

- **React 18** - UI framework
- **Vite 5** - Build tool
- **Tailwind CSS 3.3** - Styling
- **React Router v6** - Client-side routing
- **Material Symbols** - Icons

## 🎨 Design

- Purple brand theme (#802FC3)
- Glasmorphic UI components
- Smooth animations and transitions
- Fully responsive (mobile, tablet, desktop)
- Dark footer with contact information
